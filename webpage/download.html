从源代码构建

确保已安装 Android Studio 或 Gradle。

# 克隆并构建（Windows）
git clone https://github.com/HOE-Team/OpenDroidChat.git
cd OpenDroidChat
gradlew.bat build

# 构建成功后，APK 位于 app/build/outputs/apk/
