OpenDroidChat

一款为 Android 6+ 设计的开源、轻量级 LLM API 聊天客户端

 GitHub

程序介绍

技术栈

  • Kotlin — 主要编程语言
  • Jetpack Compose — 声明式 UI 框架
  • Material Design 3 (M3) — 原生 GUI 设计实现
  • Gradle — 构建工具
  • Compose Markdown — 渲染聊天中的 Markdown
  • OkHttp — HTTP 客户端

特性

  • 支持 Android 6.0(API23) 及以上
  • 系统原生、美观的 Compose + M3 界面
  • 支持 Markdown 消息渲染（代码块、标题、链接等）
  • 轻量级，易于安装和使用；支持多种 LLM API 并对 API Key 进行加密保护

预置的 LLM 提供商

  • OpenAI / Azure OpenAI
  • Google Gemini
  • DeepSeek
  • Dashscope (阿里云百炼)
  • Claude
